# Lab 0 — Basics

## 📝 Overview
This lab introduces foundational Python concepts through five small programs focused on input/output, string manipulation, and basic arithmetic.

## 🧠 Concepts Covered
- Input and output
- Strings and formatting
- Basic arithmetic
- Conditionals

## 📂 Files
- `einstein.py` — E = mc² calculator
- `faces.py` — converts :) and :( to emoji
- `indoor.py` — converts text to lowercase
- `playback.py` — replaces spaces with ellipses
- `tip.py` — calculates a restaurant tip
