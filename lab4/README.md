# Lab 4 — Libraries and Exceptions

## 📝 Overview
This lab introduces Python libraries, exception handling, and more advanced string and input processing.

## 🧠 Concepts Covered
- Using external libraries (pyfiglet, requests)
- Exception handling (try/except)
- String formatting and joining
- Input validation

## 📂 Files
- `adieu.py` — prints names in a grammatically correct list
- `bitcoin.py` — fetches Bitcoin price using an API
- `emojize.py` — converts text to emoji using the emoji library
- `figlet.py` — renders text using ASCII art fonts
- `game.py` — implements a number guessing game
- `professor.py` — math quiz with error handling
