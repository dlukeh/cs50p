# Lab 2 — Loops and String Processing

## 📝 Overview
This lab focuses on loops, string manipulation, and basic algorithmic thinking through five small Python programs.

## 🧠 Concepts Covered
- Loops (for/while)
- String iteration and transformation
- Input validation
- Basic algorithms

## 📂 Files
- `camel.py` — converts camelCase to snake_case
- `coke.py` — simulates a vending machine
- `nutrition.py` — looks up calories from a dictionary
- `plates.py` — validates license plate formats
- `twttr.py` — removes vowels from text
