# Lab 3 — Data Structures and Validation

## 📝 Overview
This lab focuses on dictionaries, lists, input validation, and simple parsing tasks.

## 🧠 Concepts Covered
- Dictionaries and key lookup
- List iteration
- Input validation
- String parsing and formatting

## 📂 Files
- `fuel.py` — converts fractions to fuel gauge percentages
- `grocery.py` — tallies grocery items using a dictionary
- `outdated.py` — parses and normalizes date formats
- `taqueria.py` — calculates order totals from a menu dictionary
