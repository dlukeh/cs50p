# Lab 1 — Conditionals and Strings

## 📝 Overview
This lab focuses on conditionals, string manipulation, and basic program flow through five small Python programs.

## 🧠 Concepts Covered
- Conditionals (if/elif/else)
- String methods and slicing
- Input validation
- Basic logic and branching

## 📂 Files
- `bank.py` — returns a greeting value based on input
- `deep.py` — checks if input matches “42”
- `extensions.py` — determines file type from extension
- `interpreter.py` — evaluates simple arithmetic expressions
- `meal.py` — converts time to meal periods
